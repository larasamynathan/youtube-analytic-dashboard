\# YouTube Channel Performance and Engagement Analytics Dashboard



\## Milestone 1



\### Objective

To connect with the YouTube Data API and extract basic channel information.



\### Features Implemented

\- Connected to YouTube Data API

\- Extracted channel details

\- Displayed channel name

\- Displayed subscriber count

\- Displayed total views

\- Displayed total video count

\- Basic Streamlit user interface



\### Technologies Used

\- Python

\- Streamlit

\- YouTube Data API

\- Pandas



\### How to Run the Project



1\. Install the required libraries:

&nbsp;  pip install -r requirements.txt



2\. Run the application:

&nbsp;  streamlit run streamlit\_app.py
